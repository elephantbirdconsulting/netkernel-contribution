KBO as Linked Open Data

Linked Open Data is data published on the web in such a way that (Bizer, 2009)

  • it is machine-readable
  • its meaning is explicitly defined
  • it is linked to other external datasets
  • can in turn be linked to from external datasets, meaning that every entity of the KBO dataset has a http uri, where information on this entity can be found.

Some examples:

  • http://data.kbodata.be/organisation/2_146_784_105#id for the organisation with KBO code 2.146.784.105
  • http://data.kbodata.be/organisation/2_198_544_392#id for the organisation with KBO code 2.198.544.392

For more info see the technical documentation.

machine readable

Using this identifier one can retrieve, next to a html page aimed at humans, the following machine readable formats: rdf/xml, json-ld, turtle, n-triples.

For more info see the technical documentation.

the meaning is explicitly defined

All entities are described using classes and properties from well-known standard vocabularies such as: org for describing organisations, regorg for registered organisations, foaf for describing persons, vcard for contact information, core location for describing locations, ... All these classes and properties have their own http address and can be looked up to see how they are defined and which semantics they carry.

For more info see the technical documentation.

linked to other datasets

Not yet, but we are working on this.

linked to from other datasets

We know already of one other linked open data application that will point to the identifiers used in this dataset. This is in fact one of the main purposes of publishing linked open data. The authoritive maintainer and source of a dataset (e.g. KBO) publishes the data and all other applications can point to the used identifiers, freeing other applications of maintaining this information by themselves. To aid people in pointing to the correct identifiers we offer a reconciliation service where you can send a label (string) to and the service responds with the corresponding identifier.
